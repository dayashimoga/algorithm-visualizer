📚 Algorithm Complexity

Bubble Sort

O(n²) time. Repeatedly swaps adjacent elements. Simple but slow.

Quick Sort

O(n log n) avg. Divides array around a pivot. Very fast in practice.

Merge Sort

O(n log n) guaranteed. Divides and merges. Stable but uses extra memory.

A* Search

Optimal pathfinding using heuristics. Faster than Dijkstra's with admissible heuristic.
